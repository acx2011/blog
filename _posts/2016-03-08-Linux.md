**摘要**: linux 中有些命令总是记不住。可能就是也不怎么常用，一年半载用一次的时候查一下然后解决了最后就撂过了。这里把自身常用的一些命令做个记录，以后还是记不住就来这里查。
{:.message}

## 磁盘满时，如何定位并删除响应文件

```shell
# 先看一下磁盘使用情况
df -h

# cd 进入那个比较满的目录，然后执行下面的命令，可以列出所有文件夹的大小
du -shx *

# 进入那个比较满的目录，找到那个占用空间较大的文件
ll -h

# 一般应该是某个日志文件，没啥用了清空即可
> filename
```
